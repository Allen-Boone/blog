### 2️⃣ 全新精简主题（Minimal Theme）

#### 精简主题的特点
- ✨ **极简设计** - 减少视觉装饰
- 🎯 **高度清爽** - 柔和的颜色和阴影
- 📱 **专注内容** - 减少干扰元素
- 💬 **舒适阅读** - 更好的排版

#### 精简主题的CSS变量
```css
body.minimal-mode {
    --bg-primary: #ffffff           /* 纯白背景 */
    --bg-secondary: #f9f9f9         /* 超浅灰背景 */
    --text-primary: #2d2d2d         /* 深灰文字 */
    --text-secondary: #727272       /* 中灰文字 */
    --text-tertiary: #a0a0a0        /* 浅灰文字 */
    --border-color: #e8e8e8         /* 超浅灰边框 */
    --border-light: #f0f0f0         /* 更浅边框 */
    --shadow-color: rgba(0,0,0,0.08)/* 极浅阴影 */
    --link-color: #0066cc           /* 深蓝链接 */
    --hover-bg: #f5f5f5             /* 浅灰hover */
}
```

#### 精简主题的优点
- 📱 字体更清晰
- 🎨 边框更细致（颜色更浅）
- 💫 阴影最小化（仅0.08透明度）
- 🌟 背景更白（纯白而非米白）
- ✨ 链接颜色更深（#0066cc）

### 3️⃣ 新主题系统

#### 主题选择菜单
```
🎨 按钮 (Header中)
    ↓
点击显示下拉菜单
    ├─ ☀️ 亮色模式 (light)
    ├─ 🌙 暗黑模式 (dark)
    └─ ✨ 精简模式 (minimal)
```

#### 主题切换机制
- `toggleThemeMenu()` - 打开/关闭菜单
- `setTheme(theme)` - 切换主题并保存
- `updateThemeOptions()` - 更新菜单显示
- `loadThemePreference()` - 加载保存的主题

#### 主题偏好保存
- localStorage键：`theme-preference`
- 保存的值：`"light"` / `"dark"` / `"minimal"`
- 自动恢复：页面加载时调用loadThemePreference()
