## 💾 主题偏好保存

### 自动保存机制
```
用户切换主题
    ↓
setTheme() 函数执行
    ↓
localStorage.setItem('theme-preference', theme)
    ↓
主题偏好保存到浏览器
    ↓
页面刷新 → 自动加载保存的主题
    ↓
关闭浏览器 → 下次打开仍是该主题
```

### 验证方法
```
1. 打开浏览器开发者工具 (F12)
2. 进入 Application 标签
3. 找到 LocalStorage
4. 查看 'theme-preference' 值
   → "light" / "dark" / "minimal"
```

## 🐛 故障排除

### 问题：主题不切换
**解决方案**：
1. Ctrl+Shift+R 强制刷新
2. 清除浏览器缓存
3. 检查浏览器是否禁用JavaScript

### 问题：菜单不出现
**解决方案**：
1. 检查是否点击到按钮
2. 刷新页面重试
3. 在浏览器console检查是否有错误

### 问题：主题不保存
**解决方案**：
1. 检查是否在无痕模式下
2. 检查localStorage是否被禁用
3. 检查浏览器隐私设置
