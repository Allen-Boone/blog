## 🎨 新增精简主题（Minimal Theme）

### 什么是精简主题？
精简主题是一个极简设计的主题，强调：
- **干净** - 最少化的视觉装饰
- **清爽** - 柔和的颜色和阴影
- **专注** - 减少干扰，突出内容
- **舒适** - 更舒适的阅读体验

### 精简主题的特点

#### 色彩方案
| 元素 | 颜色 | 说明 |
|------|------|------|
| 背景 | #ffffff | 纯白色背景 |
| 次背景 | #f9f9f9 | 浅灰色背景 |
| 文字 | #2d2d2d | 深灰色文字 |
| 次文字 | #727272 | 中灰色文字 |
| 边框 | #e8e8e8 | 超浅灰边框 |
| 链接 | #0066cc | 深蓝色链接 |
| 阴影 | rgba(0,0,0,0.08) | 极浅阴影 |

#### 设计特点
- 🎨 边框颜色更浅（#e8e8e8）
- 💫 阴影更轻（仅0.08透明度）
- 📏 更多的空白和呼吸空间
- ✨ 减少渐变效果，强调简洁

## 🎯 如何切换主题

### 新的主题切换菜单
1. 打开网站，在Header找到🎨按钮
2. 点击按钮，出现下拉菜单
3. 选择要使用的主题：
   - ☀️ **亮色模式** - 标准的蓝色主题（默认）
   - 🌙 **暗黑模式** - 深黑色主题（保护眼睛）
   - ✨ **精简模式** - 极简设计主题（最清爽）

### 主题选择的表示
- 当前选中的主题右侧会显示 ✓ 勾选符号
- 选项背景会变成蓝色渐变色
- 菜单关闭后不需要刷新，主题即时应用

## 📊 三种主题对比

### 亮色模式（Light Mode）
✨ **特点**：
- 蓝色为主色调（#1890ff）
- 白色背景（#ffffff）
- 深色文字（#333333）
- 现代设计风格
- 适合白天使用

🎯 **适用场景**：
- 白天使用
- 办公环境
- 需要高对比度

---

### 暗黑模式（Dark Mode）
🌙 **特点**：
- 深黑色背景（#1a1a1a）
- 浅灰色文字（#e8e8e8）
- 浅蓝色链接（#4da6ff）
- 舒适的夜间体验
- 降低眼睛疲劳

🎯 **适用场景**：
- 夜间使用
- 暗光环境
- 保护眼睛
- 长时间阅读

---

### 精简模式（Minimal Mode）
✨ **特点**：
- 极简设计
- 纯白色背景（#ffffff）
- 柔和的灰色文字（#2d2d2d）
- 深蓝色链接（#0066cc）
- 极浅的阴影（仅0.08透明度）

🎯 **适用场景**：
- 追求简洁的用户
- 专注阅读内容
- 中等光线环境
- 喜欢极简设计

## 🔄 主题偏好保存

- ✅ 每次切换主题时自动保存到localStorage
- ✅ 刷新页面后自动恢复选择的主题
- ✅ 关闭浏览器再打开仍保留主题设置
- ✅ 不同浏览器独立保存（互不影响）

**localStorage键**：`theme-preference`  
**可能的值**：`"light"` / `"dark"` / `"minimal"`

## 🎨 CSS变量系统

每个主题都有独立的CSS变量集合，确保颜色的一致性：

**亮色模式**（:root）
```css
--bg-primary: #ffffff
--text-primary: #333333
--link-color: #1890ff
/* ... 共10个变量 */
```

**暗黑模式**（body.dark-mode）
```css
--bg-primary: #1a1a1a
--text-primary: #e8e8e8
--link-color: #4da6ff
/* ... 共10个变量 */
```

**精简模式**（body.minimal-mode）
```css
--bg-primary: #ffffff
--text-primary: #2d2d2d
--link-color: #0066cc
/* ... 共10个变量 */
```

## 🚀 技术亮点

### 1. 统一的主题系统
- 使用CSS变量管理所有颜色
- 一个变量修改，整个应用自动更新
- 便于未来添加更多主题

### 2. 平滑的过渡
- 所有颜色变化使用 `transition: 0.3s`
- 无闪烁、无突兀切换
- 用户体验流畅

### 3. 完整的组件覆盖
- 40+个UI组件都有主题适配
- 从按钮到表格，从模态框到评论区
- 确保一致的视觉体验

### 4. 性能优化
- 纯CSS实现，零JavaScript运算
- localStorage快速加载
- 不影响页面性能

## 🔧 自定义主题

如果想修改某个主题的颜色，只需编辑CSS中的CSS变量：

**修改亮色模式的主色**
```css
:root {
    --link-color: #ff6b6b;  /* 改为红色 */
}
```

**修改精简模式的文字颜色**
```css
body.minimal-mode {
    --text-primary: #1a1a1a;  /* 改为更深的黑 */
}
```
