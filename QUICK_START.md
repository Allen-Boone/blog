## 🎓 给开发者的提示

想要修改颜色？编辑 `:root` 块中的CSS变量：

```css
:root {
    --bg-primary: #ffffff;    /* ← 改这里 */
    /* ... */
}

body.dark-mode {
    --bg-primary: #1a1a1a;    /* ← 或这里 */
    /* ... */
}
```

想要添加新主题？创建新类：

```css
body.sepia { --bg-primary: #f5e6d3; /* ... */ }
```
