# 🔐 NodeSeek 论坛 - 管理员登录指南

## 默认管理员账号

```
账号: admin
密码: admin123
```

## 访问管理面板

1. **打开网站** → 点击右上角 "管理员" 按钮
2. **输入凭证** → 使用上述账号和密码登录
3. **进入管理面板** → 点击用户菜单中的 "管理面板"

## 管理功能

### 👥 用户管理
- 查看所有注册用户
- 删除用户账号
- 显示用户ID、用户名、邮箱

### 📝 帖子管理
- 查看所有帖子
- 删除不适当的帖子
- 查看浏览量和作者信息
- 显示发帖时间和分类

### 💬 评论管理
- 查看所有评论
- 删除不当评论
- 查看评论所属帖子
- 显示评论者和发表时间

### ⚙️ 系统设置
- **统计数据**
  - 总用户数
  - 总帖子数
  - 总评论数
  - 总浏览数
  
- **危险操作**
  - 清空所有数据（二次确认）

## 功能说明

### 删除操作
- 删除用户：会保留该用户发布的帖子和评论
- 删除帖子：同时删除所有相关评论
- 删除评论：只删除该条评论

### 数据持久化
所有数据都保存在浏览器的 LocalStorage 中，刷新页面后数据不会丢失

### 双重确认
危险操作（如清空所有数据）需要二次确认，防止误操作

## 安全建议

1. 修改默认管理员密码（在 index.html 中修改 ADMIN_PASSWORD）
2. 添加更多管理员账号
3. 实现数据备份功能
4. 添加操作日志记录
5. 使用后端认证和数据库

## 修改管理员密码

打开 `index.html`，找到以下代码行：

```javascript
const ADMIN_USERNAME = 'admin';
const ADMIN_PASSWORD = 'admin123';
```

修改 `ADMIN_PASSWORD` 的值即可更改密码。

## 添加新管理员

在初始化代码中修改 admins 数组：

```javascript
admins = [
    { id: 1, username: 'admin', password: 'admin123', name: '超级管理员' },
    { id: 2, username: 'manager', password: 'manager123', name: '内容管理员' }
];
```
